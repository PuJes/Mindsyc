# 示例：视频建议中建议的英语学习方法 → 融合进原有英语口语学习方法

> 目标：只用 **1 个英语学习方法视频**，演示从“要点卡片 → 新导图 → 合并进旧框架 → 最终脑图”的全流程。

## 1) 旧框架脑图（原有知识体系）

```
  └─ 练习计划
      ├─ 听-模-读-说
      └─ 每日打卡
```

## 2) 输入视频（示例）

- **视频A**：How to Learn English Effectively（英语学习方法）

### 系统针对输入视频生成的“要点卡片”示例

- **卡片1｜学习循环：听→模仿→复述**（00:45）
  - 摘要：先听 1–2 遍抓主旨，再逐句模仿口型与节奏，最后不看稿复述 30–60 秒；每轮不超过 5 分钟。
- **卡片2｜影子跟读（Shadowing）**（02:10）
  - 摘要：选择 10–20 秒素材，跟随原音不间断模仿；先音准节奏，再追求语调自然。
- **卡片3｜词块（Chunks）优先**（03:20）
  - 摘要：以“功能词块”替代单词背诵，如 “Could you…?” “Would it be possible…?”
- **卡片4｜连读/弱读/重读是效率杠杆**（05:05）
  - 摘要：先掌握常见连读与功能词弱读，配合“重读关键信息”，可显著提升可懂度。
- **卡片5｜70/30 输入输出比**（06:40）
  - 摘要：每次练习 70% 输入（听/模仿/跟读），30% 输出（复述/即兴 1 分钟）。

## 4) 系统“对齐并合并”建议

- **合并建议**
  1. 将【节奏与发音/连读·弱读·重读】→ 合并到 **发音 & 音变 / 连读·弱读**（置信度 0.86）
  2. 将【语言单位/词块（Chunks）】→ 合并到 **高频句型** 下，新增二级 **句型构件：词块**（0.78）
- **新增建议** 3) 将【学习循环/听→模仿→复述】→ 新增到 **练习计划** 下的 **方法论**（0.74） 4) 将【技术动作/影子跟读】→ 新增到 **练习计划/听-模-读-说** 的子节点 **影子跟读**（0.81）
- **忽略建议** 5) 忽略【励志型建议：保持动力】（与旧图“每日打卡”重复，0.42）

> 交互：在“对齐面板”中逐条勾选（A=合并 / N=新增 / I=忽略）；悬停可预览对应视频时间戳片段。

### 冲突处理（若存在）

- 旧图在 **练习计划** 写着“优先背单词表”，而新导图强调“词块优先”。
  - 处理：选择 **并存**，并在 **高频句型** 下添加注记——“初级：高频词块；备考：词表+词块结合”。
